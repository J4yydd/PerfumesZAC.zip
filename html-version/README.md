# Parfum.zip - Versión HTML/CSS/JS

Esta es la versión HTML pura del proyecto, sin dependencias de React o Next.js.

## Estructura

- `index.html` - Página principal
- `shop.html` - Tienda con filtros
- `product.html` - Detalle de producto
- `cart.html` - Carrito de compras
- `contact.html` - Página de contacto
- `js/` - Archivos JavaScript
  - `products.js` - Datos de productos
  - `store.js` - Configuración de la tienda
  - `utils.js` - Utilidades (formato de moneda)
  - `cart.js` - Lógica del carrito
  - `app.js` - Funciones comunes
- `css/` - Estilos adicionales (si es necesario)

## Características

- ✅ Diseño idéntico al proyecto original
- ✅ Carrito con localStorage
- ✅ Filtros y búsqueda en la tienda
- ✅ Selector de tamaño (5ml/10ml) en productos
- ✅ Checkout por WhatsApp
- ✅ Responsive design
- ✅ Sin dependencias de build

## Cómo usar

Simplemente abre `index.html` en tu navegador. No requiere servidor ni instalación.

## Notas

- Usa Tailwind CSS via CDN para mantener el mismo diseño
- El carrito se guarda en localStorage del navegador
- Todos los enlaces son relativos, funciona sin servidor
